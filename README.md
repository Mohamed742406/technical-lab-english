# Lab English 🔬

تطبيق تعلم الإنجليزية لفني المختبرات — مبني بـ React + TypeScript + Vite.

## التشغيل محلياً

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

## التقنيات المستخدمة

- React 18 + TypeScript
- Vite + PWA (يعمل بدون إنترنت)
- Tailwind CSS + shadcn/ui
- Web Speech API (نطق + تسجيل صوت)
